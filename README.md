# 🌱 Smart Crop Recommendation System

## Overview
This project predicts the most suitable crop based on soil nutrients (N, P, K, pH) and environmental conditions like temperature, humidity, and rainfall.

It uses machine learning techniques to help in better crop selection.

---

## Features
- Predict crop based on input values
- Uses multiple ML models
- Simple and easy-to-use interface

---

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

---

## How to Run
1. Install required libraries:
   pip install -r requirements.txt

2. Run the application:
   streamlit run app.py
